API:
* создать машину - POST /api/v1/cars
пример тела запроса:

```json
{
  "id": 1,
  "color": "black",
  "numberOfWheels": 3
}
```


* найти машину по id - GET /api/v1/cars/<id>
* найти все машины - GET /api/v1/cars/
* удалить машину по id - DELETE /api/v1/cars/<id>
